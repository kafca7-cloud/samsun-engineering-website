# 홈페이지 제작 계획

## 1단계: 기획 정리
- 핵심 메시지
  - CAT 육해상 공식 딜러
  - 35년간 대기업·공기업과 쌓은 신뢰와 실적
  - 24시간 긴급 대응 체계
- 주요 방문자
  - 대기업 / 공기업 비즈니스 파트너
  - 엔드유저(고객)
- 강조 콘텐츠
  - 인증 / 기술력
  - 제품 / 서비스
  - 최적의 파트너 (3대 강점 + 고객사)

## 2단계: 사이트 구조
- Home
- 회사소개
- 제품/서비스
- 최적의 파트너 (기존 납품실적 → 3대 강점 + 고객사로 변경)
- 인증/기술력
- 문의하기

## 3단계: 현재 진행 상태
- `index.html` 기본 구조 생성 완료
- `styles.css` 디자인 색상 및 레이아웃 적용
- `brand/logo/` 폴더에 로고 파일 연결 완료
- CAT 딜러/35년 실적 배지 제거
- 납품실적 섹션 삭제 및 최적의 파트너 섹션으로 대체
- 고객사 로고 폴더 생성 (brand/logo/clients/)

## 4단계: 다음 진행
1. 고객사 로고 이미지 파일 추가 (brand/logo/clients/ 폴더에 PNG 파일)
2. 실제 연락처/이메일/주소 정보 재확인
3. 추가로 반영할 콘텐츠 확인
4. 모바일/반응형 최종 테스트

## 5단계: 향후 작업
- 모바일/반응형 테스트
- 문의 폼 서버 연동 또는 메일 발송 기능 추가
- 호스팅 연결 및 배포
